# Producto

Taller Paramétrico es una aplicación educativa de escritorio orientada a estudiar navegación 2D, campos vectoriales y métodos numéricos sin depender de backend ni red.

## Política oficial

- Plataforma oficial: Windows 10/11 x64.
- Toolchain oficial: CMake + Ninja + MinGW64 + wxWidgets local.
- Naturaleza del producto: aplicación autocontenida y educativa.
- Estado del editor de campo vectorial: experimental / V2.
- Prioridad actual: endurecimiento, refactor serio y mejora del módulo teórico.

## Valor del producto

El proyecto cumple dos objetivos al mismo tiempo:

1. servir como software educativo para visualizar conceptos de ingeniería;
2. servir como proyecto de estudio de arquitectura, documentación y refactor en C++.


## Estado actual del refactor

- teoría reforzada con ejercicios y figuras estáticas por tema;
- documentación viva compactada;
- `MainWindow` dividido por responsabilidades;
- paneles principales extraídos;
- trazabilidad básica y avanzada configuradas desde el arranque.
